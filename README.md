# 📚 英语单词学习平台

一个功能完善的英语单词学习平台，采用 React + Vite 构建，融合科学记忆方法，支持虚拟键盘输入、多词本管理、错词本、遗忘曲线复习和成就徽章系统。

## ✨ 功能特性

### 🎯 学习系统
- **双模式学习** - 学习模式和考试模式自由切换
  - 学习模式：完整显示单词、音标、释义和例句
  - 考试模式：隐藏单词，考验拼写能力
- **虚拟键盘** - 屏幕模拟键盘，支持点击和物理键盘双操作
- **单词发音** - 支持语音合成，自动朗读单词
- **智能提示** - 考试模式下提供单词长度和例句提示
- **连续正确追踪** - 记录连续正确次数，激发学习动力

### 📊 进度管理
- **每日目标设置** - 自定义每日学习单词数量（1-100个）
- **学习进度跟踪** - 实时显示今日学习进度
- **自动打卡** - 完成每日目标自动打卡，获得额外积分奖励
- **数据统计** - 已学单词、词库数量、正确率、连续正确等全方位统计
- **预计完成天数** - 根据学习进度智能计算预计完成时间

### 📚 词库管理
- **多词本支持** - 支持创建和管理多个词本
- **词本切换** - 学习时可选择不同词本或全部词本
- **独立进度** - 每个词本保存独立的学习进度
- **批量操作** - 支持批量导入、编辑、删除单词
- **搜索过滤** - 快速查找词库中的单词
- **导入导出** - 支持 JSON 和 CSV 格式的词库导入导出

### ❌ 错词本
- **自动收集** - 考试模式答错的单词自动加入错词本
- **错误统计** - 记录每个错词的错误次数和错误时间
- **错词复习** - 专门复习错误单词，针对性提升

### 🧠 遗忘曲线复习
- **科学记忆** - 基于艾宾浩斯遗忘曲线算法
- **智能复习** - 自动计算最佳复习时间
- **间隔重复** - 根据掌握程度调整复习间隔
- **等级追踪** - 5个记忆等级，逐步巩固记忆

### 🏆 成就系统
- **徽章奖励** - 8种成就徽章，激励持续学习
- **积分系统** - 完成学习目标获得积分
- **自动打卡奖励** - 完成目标自动打卡获得额外积分
- **成就解锁** - 达成特定条件自动解锁徽章

### 💾 数据管理
- **本地存储** - 所有数据保存在浏览器本地存储
- **数据备份** - 支持一键备份学习进度和词库
- **数据恢复** - 支持从备份文件恢复数据
- **自动保存** - 实时保存学习进度，防止数据丢失

### 🎨 现代化界面
- **渐变色设计** - 美观的渐变色背景
- **毛玻璃效果** - 半透明毛玻璃卡片效果
- **流畅动画** - 平滑的页面切换和交互动画
- **庆祝特效** - 完成目标时的纸屑庆祝效果
- **响应式设计** - 适配不同尺寸屏幕

## 📖 使用说明

### 1. 首页概览

首页显示整体学习情况和快速导航：

- **统计卡片** - 查看已学单词、词库数量、正确率、连续正确等
- **快速入口** - 一键跳转到学习、复习、词库管理等功能
- **每日目标** - 查看今日学习目标和完成进度
- **数据管理** - 备份和恢复学习数据

### 2. 学习模式

学习页面提供学习和考试两种模式：

#### 学习模式
- 完整显示单词、音标、释义和例句
- 使用虚拟键盘或物理键盘输入单词
- 点击🔊按钮可听单词发音
- 可按任意字母键切换到下一个单词
- 适合初学者记忆单词

#### 考试模式
- 只显示释义和音标，隐藏单词
- 需要自己拼写完整单词
- 答对获得2积分，答错重置连续正确
- 考试模式答错的单词自动加入错词本
- 可查看提示（单词长度、例句）
- 适合测试学习效果

### 3. 词库管理

#### 添加单词
- 点击"添加单词"按钮
- 输入单词、释义、例句和音标（可选）
- 选择所属词本
- 保存到词库

#### 导入词库
- 支持导入 txt 和 CSV 格式


#### 导出词库
- 选择要导出的词本
- 选择导出格式（JSON 或 CSV）
- 点击导出按钮下载文件

### 4. 错词本

- 查看所有答错的单词
- 显示每个单词的错误次数
- 支持删除错词（从错词本移除）
- 建议定期复习错词本

### 5. 遗忘曲线复习

- 查看需要复习的单词列表
- 单词按复习优先级排序
- 完成复习后自动更新复习时间
- 根据记忆等级调整下次复习时间

### 6. 徽章成就

查看已获得的徽章和待解锁条件：

| 徽章 | 名称 | 条件 |
|------|------|------|
| 🌟 | 初学者 | 完成第一个单词 |
| 📚 | 起步者 | 学习10个单词 |
| 🎯 | 进阶者 | 学习50个单词 |
| 🏆 | 学者 | 学习100个单词 |
| 👑 | 单词大师 | 学习200个单词 |
| 💯 | 完美一天 | 连续答对20个题 |
| 🔥 | 坚持不懈 | 连续学习7天 |
| ⭐ | 专家 | 正确率达到90% |

| `mistakeBook` | 错词本数据 | Array |
| `studyProgress` | 各词本学习进度 | Object |
| `dailyGoal` | 每日学习目标 | Number |
| `points` | 用户积分 | Number |
| `checkInHistory` | 打卡历史记录 | Array |
| `currentBook` | 当前选中词本 | String |

## ❓ 常见问题

### Q: 数据会丢失吗？
A: 数据存储在浏览器本地，清除浏览器数据会导致丢失。建议定期使用备份功能。

### Q: 可以同步多设备吗？
A: 目前不支持云端同步，但可以通过导出/导入词库和备份文件实现数据迁移。

### Q: 为什么听不到发音？
A: 请确保浏览器支持 Web Speech API，并检查系统音量设置。

### Q: 如何导入大量单词？
A: 使用 CSV 或 txt 格式批量导入

### Q: 可以自定义每日目标吗？
A: 可以，在首页点击"设置目标"按钮，设置1-100之间的每日单词数。
