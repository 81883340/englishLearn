# 英语单词学习平台

一个现代化的英语单词学习平台，使用 React + Vite 构建，支持虚拟键盘输入、自定义词库、学习进度保存和徽章奖励系统。

## 功能特性

- ✨ **虚拟键盘输入** - 屏幕显示模拟键盘，支持点击和物理键盘双操作
- 📚 **自定义词库** - 支持添加、编辑、删除、搜索单词
- 💾 **本地存储** - 自动保存词库、学习进度和已获得的徽章
- 📤 **导入导出** - 支持词库 JSON 格式的导入导出
- 🏆 **徽章系统** - 8种不同成就徽章，激励持续学习
- 📊 **数据统计** - 实时显示学习进度、正确率、连续正确等数据
- 🎨 **现代化界面** - 渐变色背景、毛玻璃效果、流畅动画

## 快速开始


### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 使用说明

### 1. 首页
- 查看学习统计数据
- 快速导航到各个功能页面
- 查看学习进度和成就概览

### 2. 学习模式
- 显示单词及其释义和例句
- 使用虚拟键盘或物理键盘输入单词拼写
- 正确拼写自动进入下一个单词
- 支持显示提示和查看答案
- 连续正确会有特殊奖励效果

### 3. 词库管理
- 添加新单词（单词、释义、例句）
- 编辑已有单词
- 删除单词
- 搜索功能
- 导入/导出词库（JSON格式）

### 4. 徽章成就
- 查看已获得的徽章
- 查看待解锁的徽章条件
- 查看徽章获取进度

## 徽章列表

| 徽章 | 名称 | 条件 |
|------|------|------|
| 🌟 | 初学者 | 完成第一个单词 |
| 📚 | 起步者 | 学习10个单词 |
| 🎯 | 进阶者 | 学习50个单词 |
| 🏆 | 学者 | 学习100个单词 |
| 💯 | 完美一天 | 连续答对20个题 |
| 🔥 | 坚持不懈 | 连续学习7天 |
| 👑 | 单词大师 | 学习200个单词 |
| ⭐ | 专家 | 正确率达到90% |

## 技术栈

- React 18
- Vite
- confetti（庆祝动画）
- react-hot-toast（消息提示）
- LocalStorage（本地数据存储）

## 数据存储

所有数据都保存在浏览器本地存储中：
- `wordLibrary` - 单词库
- `learnedWords` - 已学习的单词ID列表
- `englishProgress` - 学习进度和统计数据

## 快捷键

在学习模式中：
- `A-Z` - 输入字母
- `空格` - 输入空格
- `Backspace` - 删除上一个字符
- `Enter` - 进入下一个单词（显示答案后）
